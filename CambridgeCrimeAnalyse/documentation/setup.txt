## ğŸ› Troubleshooting

### "ModuleNotFoundError: No module named 'pandas'"
**Solution**: Install requirements
```bash
pip install -r requirements.txt
```

### "FileNotFoundError: [Errno 2] No such file or directory: '../data/raw/merged_file.csv'"
**Solution**:
- Check that `merged_file.csv` is in `data/raw/` folder
- Make sure you're running from `scripts/` folder: `cd scripts` then `python main.py`

### "Permission denied" when creating output folders
**Solution**: Run your terminal as administrator

### Import errors between modules
**Solution**: Make sure all Python files are in the same `scripts/` folder
